## Commandes Linux essentielles (vérifiées et triées)

### 📂 Navigation et Exploration
- `pwd` : Affiche le chemin complet du dossier où vous vous trouvez (*Print Working Directory*).
- `cd` : Change de répertoire.
- `cd ..` : Remonte d'un niveau.
- `cd ~` : Retourne au dossier personnel.
- `ls` : Liste le contenu d'un dossier.
- `ls -a` : Affiche aussi les fichiers cachés.
- `ls -h` : Affiche les tailles de fichiers de façon lisible (Ko, Mo).

### 🛠️ Manipulation de fichiers et dossiers
- `mkdir` : Crée un nouveau dossier.
- `touch` : Crée un fichier vide.
- `cp` : Copie un fichier ou un dossier.
- `cp -r` : Copie un dossier et tout son contenu.
- `mv` : Déplace ou renomme un fichier/dossier.
- `rm` : Supprime un fichier.
- `rm -rf` : Supprime un dossier et tout son contenu (attention, c'est définitif !).
- `ln -s` : Crée un lien symbolique (un raccourci).

### 📄 Lecture et Édition
- `cat` : Affiche tout le contenu d'un fichier dans le terminal.
- `less` : Lit un fichier page par page (très pratique pour les gros fichiers).
- `head` / `tail` : Affichent les 10 premières ou dernières lignes d'un fichier.
- `nano` / `vi` : Éditeurs de texte directement dans le terminal.

### 🔍 Recherche et Filtrage
- `grep` : Recherche un mot ou une phrase dans un fichier ou un flux de texte.
- `find` : Cherche des fichiers selon des critères (nom, date, taille).
- `locate` : Trouve rapidement un fichier via une base de données indexée.
- `which` : Affiche le chemin de l'exécutable d'une commande.

### 🔐 Permissions et Utilisateurs
- `chmod` : Modifie les permissions d'un fichier (lecture, écriture, exécution).
- `chown` : Change le propriétaire d'un fichier ou dossier.
- `sudo` : Exécute une commande avec les droits d'administrateur.

### 🖥️ Système et Ressources
- `top` ou `htop` : Affichent les processus en cours et l'utilisation de la RAM/CPU.
- `df -h` : Affiche l'espace disque disponible sur les partitions.
- `du -sh *` : Affiche la taille des dossiers et fichiers du répertoire actuel.
- `history` : Affiche la liste des dernières commandes tapées.
